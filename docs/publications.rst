Publications
============

The following publications have used TrajPy for trajectory analysis and classification.
If you use TrajPy in your research, please cite the methodological paper and the software DOI listed below.

Citing TrajPy
-------------

Please cite the methodological paper and the software DOI in any academic work that uses TrajPy:

**Methodological paper**

   Moreira-Soares, M., Mossmann, E., Travasso, R. D. M., and Bordin, J. R.
   *TrajPy: empowering feature engineering for trajectory analysis across domains.*
   Bioinformatics Advances, Volume 4, Issue 1, 2024, vbae026.
   `doi:10.1093/bioadv/vbae026 <https://doi.org/10.1093/bioadv/vbae026>`_

**Software DOI**

   Mauricio Moreira and Eduardo Mossmann.
   *phydev/trajpy: TrajPy 1.3.1.*
   Zenodo, 2020.
   `doi:10.5281/zenodo.3978699 <https://doi.org/10.5281/zenodo.3978699>`_
